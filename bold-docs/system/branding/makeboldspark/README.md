# Make Bold Spark Brand Package

This package is the reusable branding kit for Make Bold Spark projects, products, demos, and websites.

## Contents

- `brand-guide.md` - full brand guidance for strategy, voice, identity, color, typography, layout, and usage.
- `brand-tokens.json` - machine-readable color, type, radius, and shadow tokens.
- `css/makeboldspark-brand.css` - web-ready CSS variables, typography, buttons, cards, badges, and utility classes.
- `assets/logo/` - SVG source logos and marks.
- `assets/favicon/` - favicon SVG, PNG exports, ICO file, and web manifest.
- `assets/social/` - social avatar and share image source SVGs.
- `templates/html-head.html` - copy-ready website head snippet.

## Quick Start

1. Copy `css/makeboldspark-brand.css` into a project or serve it from a shared static asset location.
2. Copy `assets/favicon/*` to the website public root or static assets directory.
3. Add the links from `templates/html-head.html` to the site layout.
4. Use `assets/logo/makeboldspark-logo-horizontal.svg` for headers and `assets/logo/makeboldspark-mark.svg` for icons, avatars, and tight UI.

## Canonical Name

Use `Make Bold Spark` in prose and page titles.

Use `MakeBoldSpark` only for code namespaces, package names, repository names, handles, domains, or compact product identifiers.
